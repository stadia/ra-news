## 🛠 기술 스택

- **Backend**: Rails 8, Ruby 3.4, PostgreSQL
- **Frontend**: Hotwire (Turbo/Stimulus), Tailwind CSS 4.2, Flowbite
- **AI**: RubyLLM with Gemini models
- **Testing**: Minitest, RuboCop, Steep (type checking)
- **Deployment**: Production at https://ruby-news.kr/
